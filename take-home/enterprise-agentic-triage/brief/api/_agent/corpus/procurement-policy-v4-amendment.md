# Procurement Policy Amendment — v4

Effective date: 2026-04-01. Owner: Chief Procurement Office.

This amendment supersedes Global Procurement Policy v3, Section 4, for all
purchase requests dated on or after the effective date. All other sections
of v3 remain in force.

## 4. Software purchases (amended)

Software purchases above $25,000 require written approval from the CTO
before the purchase order is issued. This threshold replaces the $50,000
threshold in v3 Section 4.

## 4a. Cloud services (new)

Cloud services are treated as software for approval purposes. Any cloud
service that stores customer data must hold a current SOC 2 Type II report
(see the IT Software Standards).
