# Delta — Viral Post Playbook

*Reusable for all 12 posts. Built from current LinkedIn/X algorithm research + reverse-engineering the FDE posts that already went big (Bloomberry, a16z, Qureshi, Mabrey).*

## The one-line formula
Open with a precise number OR a funny analogy → bust a myth with a taxonomy → prove each beat with one stat + one "so what" → pre-empt the "it's just a consultant" skeptic → land a coined-term reframe → close with a vivid anecdote + "save this" + a real question. Link in comments. 3–5 hashtags. Reply to everyone in the first hour.

## Hook templates (ranked)
1. **"I analyzed N" data-bomb** — precise count + one shocking number above the fold. (Bloomberry's format; highest save-rate.)
2. **Authority-borrow + contrarian** — "a16z calls it the hottest job. Palantir spent 20 years proving everyone wrong about it."
3. **Stat-shock** — "FDE jobs grew 1,165% last year. Here's the role eating both 'engineer' and 'salesperson':"
4. **Everyday analogy** — a16z's "grandma's iPhone" was the single most-shared line in the corpus.
5. **Myth-buster** — "Unpopular opinion: 90% of 'FDE' jobs posted today aren't FDEs."
6. **Litmus test** — "Debugging production at 2am on-site? You're an FDE. Hand off after the demo? You're not." (screenshot bait)

Hook rules: under ~200 chars, lead with the number or contrarian claim, no emoji/greeting before it, deliver tension or a specific promise.

## Post skeleton
HOOK (line 1–2, above fold: number + contrarian) → STAKES (one line: who/why now) → THE TURN (taxonomy or myth-bust = your organizing idea) → BODY (4–6 beats, each = CLAIM → STAT → SO-WHAT, one idea per line) → REFRAME (coined-term payoff) → VIVID ANECDOTE (one weird specific image) → CLOSE ("save this" + one real question) → 3–5 hashtags.

## Formatting rules
- One idea per line; white space between beats. Dwell time is LinkedIn's #1 signal.
- Length 1,300–2,100 chars sweet spot; hard ceiling ~2,500 (over = ~−35% reach).
- Stats: one per line, number first, then the "so what." Never stack 3 numbers in a sentence.
- Precise for credibility (1,000 posts), round for punch (1,165%, 0% quota).
- Attribute inline + light: "(per Bloomberry's 1,000-job analysis)" — no footnote block.
- LINKS IN FIRST COMMENT (LinkedIn) or reply (X). Links in body = ~60% less reach.
- Emojis sparingly as markers only, never before the hook. Hashtags 3–5 at bottom.
- First 30–60 min decisive: reply to every comment fast.

## Do / Don't
DO: lead with one big number OR one analogy (not both); coin/borrow a term; give a taxonomy or litmus test readers can self-classify with; drop one weird specific anecdote; pre-empt the "just a consultant" skeptic in-post; ask one substantive question + "save this."

DON'T: use "Agree?/Thoughts?/Comment YES/Tag someone" (2026 classifier suppresses it); write LLM cadence ("it's not X, it's Y," "not just smart — it's required"); put links in body; stack stats; exceed ~2,500 chars; post twice within 4h; leave a bare hot take without a stat on the next line.

## Lessons from the first published long-form edit (fold these into future drafts)

Captured from the human's edits to the AI-drafted FDE-os long-form (the
[published cut](https://www.linkedin.com/pulse/fastest-growing-job-ai-has-school-so-im-building-one-learns-wu-phd-uoasc/)).
These are now defaults, not afterthoughts:

- **Title leads with the data framing, not the clever framing.** "most copied job" → "**fastest-growing** job" (ties to the 1,165% stat). Foreground the live hook — "...one **that learns**."
- **Name process loops as operational steps, capitalized.** Soft verbs read as a vibe; named steps read as rigor to a technical director. `Deploy → Observe → Build → Validate → Productize → Repeat` beats "deploy → learn → build → codify." Bonus: reuse a step's word later (here, "Validate") to tie the model to the argument.
- **Frame "why now" through AI's probabilistic nature.** Deterministic software ships once; a probabilistic model is confidently wrong and must be *validated in the field* — a sharper hook than the bare 95%-stall stat (use the stat as the payoff, not the premise).
- **Tighten relentlessly.** The human cut, never padded.
