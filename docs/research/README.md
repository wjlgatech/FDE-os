# docs/research — external source vault + critical evaluations

External articles, talks, and repos that inform FDE-os — **kept separate from**
`FDE-research-synthesis.md` (which is the FDE-*domain* vault that feeds the knowledge spine).
This folder is for *methodology / landscape* sources and our critical take on them.

## Convention

- `sources/` — the saved source material, verbatim where possible, with honest provenance
  (who said it, when, where, and **whether multiple "sources" are actually the same content**).
- `*-critical-eval.md` — our evaluation: what's real vs. hype, how it aligns with the FDE-os
  mission, and what (if anything) we should build because of it.

Discipline (same as the vault's R4): a repost is **not** independent corroboration. Note it.
